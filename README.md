# Angular Test

This is just angular test

## Installation

Create .env fike with value of HASH_PASSWORD & MONGO_URL, example:
```bash
HASH_PASSWORD=xxxxxx
MONGO_URL=xxxxxx
```
install all node package

```bash
yarn install
```

## Usage
run application
```nodejs
yarn start
```
open your browser
```bash
http://localhost:3000
```
